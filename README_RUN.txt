# Інструкція запуску проекту

## 1. Встановлення залежностей

Встановіть залежності (один раз):
```
pip install -r requirements.txt
```

## 2. Запуск бекенду (FastAPI)

У терміналі виконайте:
```
python main.py
```
- Бекенд буде доступний за адресою: http://127.0.0.1:8001
- При запуску автоматично додаються стандартні слова та створюється адміністратор (admin/admin123).

## 3. Запуск фронтенду (Streamlit)

У новому терміналі виконайте:
```
streamlit run frontend/app.py
```
- Фронтенд буде доступний за адресою: http://localhost:8501

## 4. Корисні команди

- **Ініціалізація/відновлення бази та додавання стандартних слів вручну:**
  ```
  python setup_db.py
  ```
- **Зупинити всі процеси Python (бекенд):**
  ```
  taskkill /F /IM python.exe
  ```
- **Зупинити всі процеси Streamlit (фронтенд):**
  ```
  taskkill /F /IM streamlit.exe
  ```
- **Оновити залежності:**
  ```
  pip install --upgrade -r requirements.txt
  ```

## 5. Додатково
- Для входу як адміністратор використовуйте логін: `admin`, пароль: `admin123`.
- Нові слова можна додавати через інтерфейс у відповідній вкладці.
- Якщо слова не відображаються — переконайтесь, що база не пуста (запустіть `setup_db.py`).
